# Juego del Gato

### Descripción

Esta aplicación es un juego Tres en Raya (Gato) construida con **_React.js_**, enfocada en el manejo del **estado**, y la **inmutabilidad**.

### Herramientas

- _CSS_
  - ❌ _Responsive_
- _React_
  - _jsx_
  - _props_
  - Renderizado de Elementos (lista)
  - _useState (Hook)_

### Implementación

- tablero
- marcador
  - botón de reinicio de marcador
- detección de victoria/empate
- reinicio automático de la partida
